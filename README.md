[ OVERVIEW ]

WooDY is a clean, minimal, static website project built using pure HTML.
The repository centers around a single entry file — `index.html` — which
defines the structure and layout of the website.

No frameworks.
No build tools.
No heavy dependencies.

Just structured markup, organized layout, and fast performance.

Maintained under the Dev-Chandan404 GitHub account.

[ RUN LOCALLY ]

Clone the repository:

    git clone https://github.com/Dev-Chandan404/WooDY.git
    cd WooDY

Open directly in browser:

    index.html

Or run a local static server:

    npx serve .

Visit:

    http://localhost:3000

No installation required.
No compilation required.

--------------------------------------------------------------------------------

[ DEPLOYMENT ]

WooDY can be deployed instantly using:

• GitHub Pages
• Netlify
• Vercel
• Any static hosting provider

GitHub Pages setup:

1. Go to repository Settings
2. Open "Pages"
3. Select branch: main
4. Save

Your site goes live within seconds.
