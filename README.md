## ✨ About the Project

> An **elegant, minimal one-page HTML/CSS template** crafted for speed and beauty. WooDY delivers buttery-smooth scrolling and cinematic fade-in animations — with zero frameworks, zero build tools, and zero bloat.

This project proves that you don't need a heavy stack to create a stunning web experience. Just clean, semantic HTML and expressive CSS doing exactly what they were made to do.

---

## 🎯 Key Features

| | Feature | Description |
|---|---|---|
| 🎬 | **Cinematic Animations** | Smooth fade-in entrance animations for an immersive feel |
| 🖱️ | **Buttery Scrolling** | Silky smooth scroll behavior throughout the page |
| 🪶 | **Zero Dependencies** | Pure HTML & CSS — no frameworks, no npm, no build step |
| 📐 | **Semantic Structure** | Clean, well-organized and readable markup |
| 📱 | **Cross-Browser Ready** | Consistent experience across all modern browsers |
| ⚡ | **Instant Load** | No compilation, no bundling — opens straight in the browser |

## 🚀 Getting Started

No installation. No build step. Just clone and open.

```bash
# Clone the repo
git clone https://github.com/Dev-Chandan404/WooDY.git
cd WooDY

# Open directly in your browser
open index.html

# Or serve locally
npx serve .
# → http://localhost:3000
```
